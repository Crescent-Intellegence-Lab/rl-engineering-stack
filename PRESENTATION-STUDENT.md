# Student Presentation

The repository includes a student-facing English presentation on reinforcement-learning engineering tools.

- [Slides](presentation/slides.md)
- [Slide overview](presentation/montage.svg)
- [Student study guide](presentation/student-study-guide.md)
- [Presentation directory](presentation/README.md)

The deck begins with the classical environment–policy–collector–learner loop and then introduces Gymnasium, CleanRL, Stable-Baselines3, RLlib, TRL, veRL, OpenRLHF, rollout engines, training backends, and asynchronous agentic RL.

The editable PowerPoint is provided separately for manual upload.
